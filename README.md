# 2つのサイコロ

2つのサイコロの出目と和の分布を視覚的に確認できるインタラクティブツール

🔗 **公開URL**: https://dice.meetupsensei.com/

## 概要

2つのサイコロを振って、出目と合計値の分布をリアルタイムで確認できるツールです。
